- Free UML 2 sketching without limits

Vision

TinyUML scratches a personal itch. The goal is to provide a sketching tool that is

  • Open Source and free of charge
  • implemented in Java
  • small, reliable and simple to use

Project status

TinyUML is hosted on Sourceforge. The project's progress can be viewed in its own WordPress Blog, there is also a project roadmap.

The current release version is Version 0.12 ("Cait Sith"), there is a Maven-generated project website.

Documentation

There is now a Wiki for TinyUML related documents.
